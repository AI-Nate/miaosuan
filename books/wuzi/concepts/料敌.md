# 概念 · 料敌（审敌虚实·击八避六）

_吴子核心概念 · 出 [02 料敌篇](../02-料敌篇.md) · 入库 2026-06-15_

**一句话**：吴子的"料敌"是一套**系统的敌情判断法**——先看敌国的国情民俗（战略层），再看可击/当避的征候（战役层），最后看临阵可击之态（战术层）。比《孙子》的"相敌"更分层、更可操作。核心原则："审敌虚实而趋其危""见可而进，知难而退"。

## 三个层次

**① 战略层：六国之俗（知国情）**
吴起逐一分析齐、秦、楚、燕、三晋的民性、政情与军阵弱点，并给出各自破法：

| 国 | 军阵特点 | 破法（要点） |
|---|---|---|
| 齐 | 陈重而不坚（富骄、禄不均、一陈两心） | 三分之、猎其左右 |
| 秦 | 陈散而自斗（性强政严、人人好斗） | 示利诱离其将、设伏投机 |
| 楚 | 陈整而不久（地广政骚民疲） | 袭乱夺气、轻进速退、勿与久战 |
| 燕 | 陈守而不走（性悫民慎寡诈） | 触迫陵远、驰而后之 |
| 三晋 | 陈治而不用（民疲轻将薄禄） | 阻陈压之、来拒去追以倦其师 |

**② 战役层：击之八 / 避之六（定打不打）**
- **击之八（不卜可战）**：①疾风大寒疲行 ②盛夏炎热远驱 ③师久粮缺民怨 ④军资竭薪刍寡 ⑤众少地不利疫病 ⑥道远日暮解甲 ⑦将薄吏轻数惊 ⑧陈舍未定半隐半出 → "击之勿疑"。
- **避之六（不占当避）**：①地广民富 ②上爱其下 ③赏信刑察 ④任贤使能 ⑤师众兵精 ⑥四邻有援 → "避之勿疑"。
- 归结："**见可而进，知难而退**"。

**③ 战术层：临阵可击之兆**
- 来势荡荡无虑、旌旗烦乱、人马数顾 → 一可击十。
- 诸侯新会、君臣未和、沟垒未成 → 以半击倍。
- 必可击的十三态（远来未定、既食未备、半渡、险道狭路、将离士卒、心怖……）→ "选锐冲之，分兵继之，急击勿疑"。

## 与《孙子》对读

- 孙子 [行军篇](../../sunzi/09-行军篇.md)"相敌"是从战场征候推断敌情（战术层为主）；吴子"料敌"额外补上**敌国战略层的国情民俗分析**，更系统。
- "审敌虚实而趋其危" ↔ 孙子 [虚实](../../sunzi/concepts/虚实.md)"避实击虚"。
