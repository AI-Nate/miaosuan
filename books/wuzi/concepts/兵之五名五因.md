# 概念 · 兵之五名五因（战争性质论）

_吴子核心概念 · 出 [01 图国篇](../01-图国篇.md) · 入库 2026-06-15_

**一句话**：吴子把战争按**起因**和**性质**各分五类，并指出每种性质的战争各有不同的"制服之道"。这是中国兵学史上较早的**战争性质分类**，是《孙子》所没有的视角——孙子论"如何打赢"，吴子先问"这是哪一种战争"。

## 五因（战争的起因）

> "凡兵之所起者有五：一曰**爭名**，二曰**爭利**，三曰**積德惡**，四曰**內亂**，五曰**因饑**。"

1. 争名——争夺名分声望
2. 争利——争夺利益土地
3. 积德恶——道德/仇怨的长期积累
4. 内乱——内部动乱引发
5. 因饥——饥荒所迫

## 五名（战争的性质）及制服之道

> "其名又有五：一曰**義兵**，二曰**彊兵**，三曰**剛兵**，四曰**暴兵**，五曰**逆兵**。"

| 名 | 界定 | 制服之道 |
|---|---|---|
| 义兵 | 禁暴救乱 | 必以**礼**服 |
| 强兵 | 恃众以伐 | 必以**谦**服 |
| 刚兵 | 因怒兴师 | 必以**辞**服 |
| 暴兵 | 弃礼贪利 | 必以**诈**服 |
| 逆兵 | 国乱人疲而举事动众 | 必以**权**服 |

要点：不同性质的敌军要用不同手段应对——对义兵讲礼、对强兵示谦、对刚兵以言辞、对暴兵用诈、对逆兵用权变。**先判性质，再定对策**。

## 为什么重要

- 提供了"开战前先定性"的框架：自己发动的是哪种兵（关乎正当性与人心），面对的是哪种兵（关乎对策）。
- 与 [内修文德外治武备](内修文德外治武备.md) 呼应——义兵（禁暴救乱）才是吴子推崇的，与"先和""使民有耻"的政治基调一致。

## 与《孙子》对读

- 孙子几乎不谈战争的道德性质分类，更聚焦"利害算计 + 谋略"。吴子此论补上了**战争伦理/性质**的维度，是儒法兼综的体现。
