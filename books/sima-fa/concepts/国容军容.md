# 概念 · 国容军容（文武分际）

_司马法核心概念 · 出 [02 天子之义篇](../02-天子之义篇.md) · 入库 2026-06-15_

**一句话**：治国之礼（国容）与治军之法（军容）是**两套不同的行为规范，不可互相越界**——朝堂讲谦逊揖让，军中讲果决威严。互相混入则民德败坏。这是《司马法》对文武关系最独到的制度化表述。

## 核心命题

> "古者，**國容不入軍，軍容不入國**。軍容入國則民德廢，國容入軍則民德弱。"

| | 国容（治国之礼） | 军容（治军之法） |
|---|---|---|
| 场合 | 朝堂、国中 | 军中、战阵 |
| 风格 | 言文语温、恭以逊、难进易退 | 抗而立、遂而果、介者不拜、兵车不式 |
| 越界之害 | 国容入军 → 民德弱（军队失去果决） | 军容入国 → 民德废（朝堂失去礼让） |

结论："**礼与法表里也，文与武左右也**"——礼（文）与法（武）互为表里、如左右手，但各有其位，不可错置。

## 延伸与呼应

- [定爵篇](../03-定爵篇.md) 把它细化为三态："居国惠以信，在军广以武，刃上果以敏""居国和，在军法，刃上察"——同一原则在"国 / 军 / 临阵"三个层级的不同侧重。
- 与"不伐之士为上器""军旅以舒为主"等配合，构成一套**情境化的行为伦理**：在对的场合用对的规范。

## 与孙吴对读

- **司马法**：文武**分置**——国容、军容各守其域。
- **吴子**：文武**合于一身**——"[总文武者，军之将也](../../wuzi/concepts/将之五慎四机.md)"，要求将领兼具文德武略。
- 两种安排反映不同时代：司马法是封建分工（文官武将各司其职）的产物；吴子是战国"将"权独立、文武集于统帅一身的趋势。对读可见军事组织形态的演变。

## 现代迁移

- "国容不入军，军容不入国"近似现代的**情境领导/场景切换**：会议室的协商风格与危机现场的指挥风格应当不同，混用则两边都失效。
