# 概念 · 九伐之法（发禁者九）

_司马法核心概念 · 出 [01 仁本篇](../01-仁本篇.md) · 入库 2026-06-15_

**一句话**：《司马法》列出诸侯触犯的**九种罪行**及对应的九等讨伐处置——这是中国最早的"**出师有名**"法理清单，把战争的发动纳入明确的罪与罚框架（古代版的 jus ad bellum）。配合"入罪人之地"的军纪，构成完整的战争正当性 + 交战规范。

## 九伐（发禁者九）

> "會之以發禁者九：……"

| 罪行 | 处置 |
|---|---|
| 凭弱犯寡（恃强凌弱） | 眚之（削减、贬责） |
| 贼贤害民 | 伐之（征讨） |
| 暴内陵外（对内暴虐、对外侵略） | 坛之（会盟声讨、废其君） |
| 野荒民散（荒废国政、民众流散） | 削之（削减封地） |
| 负固不服（恃险不臣服） | 侵之（进逼） |
| 贼杀其亲（杀害亲族） | 正之（正其罪、治之） |
| 放弑其君（放逐或弑杀国君） | 残之（诛灭其身） |
| 犯令陵政（违令乱政） | 绝之（断绝邦交） |
| 外内乱、禽兽行（纲常尽失） | 灭之（灭其国） |

罪行越重，处置越严——从"眚（轻责）"到"灭（灭国）"层层递进，体现讨伐的比例原则。

## 配套：兴兵的程序与军纪

- **程序（出师有名）**：发现罪行 → 偏告诸侯、彰明有罪 → 告于皇天上帝、祷于神祇、造于先王 → 冢宰征师，宣告"某国为不道，征之"。战争被包装为**奉天讨罪**的正式程序，而非私斗。
- **入境军纪（占领规范）**：[仁本篇](../01-仁本篇.md)"入罪人之地，无暴神祇、无毁土功、无燔墙屋、无伐林木、无取六畜禾黍；见老幼奉归勿伤、敌伤则医药归之"——讨伐的是罪人（其君），而非平民，故占领时严禁劫掠残害。这是古代版的**交战/占领法（jus in bello）**。

## 为什么重要

- 把"战争正当性"从抽象原则（[仁本·以战止战](仁本与以战止战.md)）落实为**可操作的罪名清单 + 兴兵程序 + 军纪规范**，是先秦"礼义之兵"制度化的最高体现。
- 与吴子 [兵之五名五因](../../wuzi/concepts/兵之五名五因.md)（义/强/刚/暴/逆）相呼应，但司马法更具"周天子讨伐诸侯"的封建礼法色彩。
