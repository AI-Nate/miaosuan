# 概念 · 古者军礼与六德

_司马法核心概念 · 出 [01 仁本篇](../01-仁本篇.md)、[02 天子之义篇](../02-天子之义篇.md) · 入库 2026-06-15_

**一句话**：《司马法》保存了孙子之前的"**军礼**"传统——把战争当作有礼有节、点到为止的"君子之争"。"逐奔不过百步、成列而鼓、争义不争利"正是这套贵族军礼的写照，与孙子"[兵者诡道](../../sunzi/concepts/诡道.md)、攻其无备"形成中国兵学史上最鲜明的一次对照。

## 六德（军礼的六种德目）

> "**逐奔不過百步**，縱綏不過三舍，是以明其**禮**也；不窮不能而哀憐傷病，是以明其**仁**也；**成列而鼓**，是以明其**信**也；**爭義不爭利**，是以明其**義**也；又能舍服，是以明其**勇**也；知終知始，是以明其**智**也。"

| 行为 | 所明之德 |
|---|---|
| 逐奔不过百步、纵绥不过三舍（追败敌有限度） | 礼 |
| 不穷追无能者、哀怜伤病 | 仁 |
| 成列而鼓（等敌列好阵才击鼓开战） | 信 |
| 争义不争利 | 义 |
| 能赦免降服者（舍服） | 勇 |
| 知终知始（善始善终） | 智 |

"六德以时合教，以为民纪之道"——这六德是上古教化与军纪的根本。

## 历史意义：从"礼"到"诡"的转折

- 这套军礼属于**西周—春秋早中期**的贵族战争形态：双方堂堂正正列阵、击鼓而战、不乘人之危、不过度追杀。著名的"宋襄公不鼓不成列"即此传统的余响。
- 到春秋末—战国，战争升级为你死我活的总体战，孙子遂提出"兵者诡道""攻其无备，出其不意"——**军礼让位于诡诈**。
- 《司马法》之所以珍贵，正因为它**保存了被孙子取代的那套更古老的战争伦理**。读懂"成列而鼓"与"攻其无备"的对立，就抓住了先秦兵学的大转折。

## 配套：以礼为固、以仁为胜

[天子之义篇](../02-天子之义篇.md)："逐奔不远，纵绥不及……以礼为固，以仁为胜。既胜之后，其教可复。"——节制不仅是道德，也避免被诱伏（"不远则难诱，不及则难陷"），且胜后易于恢复教化。礼与节制在此既是伦理又是实用。

## 与孙吴对读

- **正题（司马法）**：以礼为固、成列而鼓、点到为止。
- **反题（孙子）**：[兵者诡道](../../sunzi/concepts/诡道.md)、避实击虚、穷寇可遏与否各有论。
- **合题（吴子）**：礼法兼综——既讲治国得民（近司马法），也讲料敌击虚（近孙子）。
