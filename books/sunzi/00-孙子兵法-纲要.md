# 《孙子兵法》纲要

_武经七书 · 第一书 · 全本入库 2026-06-15 · 经文源 [ctext.org](https://ctext.org/art-of-war/zh)_

> **一句话**：《孙子兵法》是中国现存最早、影响最大的兵书，13 篇约 6000 字，核心不是"怎么打赢一场仗"，而是"**如何在开战之前就让胜负已分，并以最小代价取全胜**"——慎战、先知、先胜、全胜。

## 作者与成书（背景，通行说法，未经本 wiki 双源校验）

- **传统归属**：春秋末期 **孙武**（齐国人，字长卿），以兵法十三篇见吴王阖闾，助吴破楚。
- **成书争议**：学界对作者与成书年代多有讨论（孙武其人、是否一人一时所作、是否含后学增补，均有不同观点）。
- **关键考古**：**1972 年山东临沂银雀山汉墓**同时出土《孙子兵法》与《孙膑兵法》竹简，证实二书各自独立存在，平息了"孙武即孙膑、二书为一"的旧疑。

## 十三篇结构（思想递进）

孙子十三篇并非松散语录，而有内在逻辑层次。常见的分组理解：

**第一层 · 战略总论（战前 / 庙堂）—— 打不打、值不值**
- [一 始计](01-始计篇.md)：以[五事七计](concepts/五事七计.md)庙算定胜负 → "兵者，国之大事"
- [二 作战](02-作战篇.md)：战争的经济成本 → "兵贵胜，不贵久""因粮于敌"
- [三 谋攻](03-谋攻篇.md)：[全胜](concepts/不战而屈人之兵.md)与谋攻 → "不战而屈人之兵""上兵伐谋"

**第二层 · 攻守与造势（实力 / 态势）—— 怎么形成压倒之势**
- [四 军形](04-军形篇.md)：实力的积累与攻守 → "先为不可胜""先胜而后求战"
- [五 兵势](05-兵势篇.md)：[势](concepts/势与节.md)的运用 → "[以正合，以奇胜](concepts/奇正.md)"
- [六 虚实](06-虚实篇.md)：[虚实](concepts/虚实.md)主动权 → "致人而不致于人""避实而击虚""兵形象水"

**第三层 · 战场机动与应变（行军 / 临阵）—— 具体怎么打**
- [七 军争](07-军争篇.md)：争夺先机 → "以迂为直""其疾如风…不动如山"
- [八 九变](08-九变篇.md)：通权达变 → "君命有所不受""智者之虑必杂于利害"
- [九 行军](09-行军篇.md)：处军、相敌、治军 → 三十余条"相敌"观察法
- [十 地形](10-地形篇.md)：六种地形、六种败因 → "知彼知己…知天知地"
- [十一 九地](11-九地篇.md)：九种战地心理与机动 → "投之亡地然后存""如率然"

**第四层 · 特种手段与收束（专题 / 全书总纲）**
- [十二 火攻](12-火攻篇.md)：火攻五法，并以**慎战**收束全书 → "非利不动，非得不用，非危不战"
- [十三 用间](13-用间篇.md)：情报为先 → [五间](concepts/五间.md)俱起，"反间"为本，呼应首篇"先知"

## 贯穿全书的思想主线

1. **慎战 / 重战**：兵者国之大事（首篇）→ 非危不战、亡国不可复存（火攻收尾）。全书首尾都在踩刹车，不是好战书。
2. **先知 / 庙算**：胜负在开战前的"算"里就已决定（始计），而"算"靠情报（用间），首尾呼应。
3. **先胜 / 不败**：[先为不可胜](04-军形篇.md)，立于不败之地，再求敌之可胜 → "胜兵先胜而后求战"。
4. **全胜 / 慎攻**：[不战而屈人之兵](concepts/不战而屈人之兵.md)为上，攻城为下；以"全"争天下。
5. **主动 / 致人**：[致人而不致于人](concepts/虚实.md)——永远争夺主动权，调动敌人而不被调动。
6. **变 / 因敌制胜**：兵无常势，水无常形（虚实）；[奇正](concepts/奇正.md)相生、[九变](08-九变篇.md)无穷。
7. **诡 / 诈**：[兵者诡道也](concepts/诡道.md)、兵以诈立——但"诈"服务于"算"与"全"，非为诈而诈。

## 高频金句（全书可背清单）

- 兵者，國之大事，死生之地，存亡之道，不可不察也。（始计）
- 兵者，詭道也……攻其無備，出其不意。（始计）
- 多算勝，少算不勝，而況於無算乎？（始计）
- 兵貴勝，不貴久。（作战）
- 不戰而屈人之兵，善之善者也。上兵伐謀。（谋攻）
- **知彼知己，百戰不殆。**（谋攻）
- 昔之善戰者，先為不可勝，以待敵之可勝。（军形）
- 勝兵先勝而後求戰，敗兵先戰而後求勝。（军形）
- 凡戰者，以正合，以奇勝。（兵势）
- 善戰者，致人而不致于人。（虚实）
- 兵形象水……避實而擊虛。兵無常勢，水無常形。（虚实）
- 其疾如風，其徐如林，侵掠如火，不動如山。（军争）
- 投之亡地然後存，陷之死地然後生。（九地）
- 非利不動，非得不用，非危不戰。（火攻）
