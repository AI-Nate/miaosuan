# 概念 · 五事七计

_孙子兵法核心概念 · 出 [01 始计篇](../01-始计篇.md) · 入库 2026-06-15_

**一句话**：开战前评估胜负的完整框架——「五事」是五个根本维度（战略体检表），「七计」是七项可量化的敌我对比（打分表）。先有五事的定性，再有七计的定量，合起来就是"庙算"。

## 五事（道、天、地、将、法）

| 事 | 原文界定 | 现代理解 |
|---|---|---|
| **道** | 令民與上同意也，可與之死，可與之生，而不畏危 | 政治认同 / 上下同心（民心、组织共识） |
| **天** | 陰陽，寒暑，時制也 | 天时：气候、季节、昼夜时机 |
| **地** | 遠近，險易，廣狹，死生也 | 地利：距离、险易、广狭、生死之地 |
| **将** | 智，信，仁，勇，嚴也 | 将才五德：智慧、信用、仁爱、勇敢、威严 |
| **法** | 曲制，官道，主用也 | 组织：编制、职官制度、后勤军需管理 |

> "凡此五者，將莫不聞，**知之者勝，不知者不勝**。"

## 七计（七项对比）

> "校之以計，而索其情。曰：主孰有道，將孰有能，天地孰得，法令孰行，兵眾孰強，士卒孰練，賞罰孰明，吾以此知勝負矣。"

1. 主孰有道（哪方君主更得道/得民心）
2. 将孰有能（哪方将领更有才）
3. 天地孰得（哪方占天时地利）
4. 法令孰行（哪方法令更能执行）
5. 兵众孰强（哪方兵力更强）
6. 士卒孰练（哪方训练更精）
7. 赏罚孰明（哪方赏罚更分明）

## 为什么重要

- 五事七计是《孙子》"**慎战 + 先知 + 先胜**"思想的操作化：不是上场比拼，而是**开战前先把账算清**。
- "将听吾计，用之必胜，留之；将不听吾计，用之必败，去之"——孙子甚至以是否采纳此框架作为去留条件。
- 现代迁移：可作商业竞争 / 项目立项的"尽职调查"清单——道（团队共识）、天（时机窗口）、地（赛道环境）、将（领导力）、法（组织流程）。
