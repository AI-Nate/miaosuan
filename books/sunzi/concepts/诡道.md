# 概念 · 诡道（兵者诡道也）

_孙子兵法核心概念 · 出 [01 始计篇](../01-始计篇.md) · 入库 2026-06-15_

**一句话**："**兵者，诡道也**"——用兵的本质是欺骗与示形，靠制造错误认知调动敌人，最终"攻其无备，出其不意"。这是孙子方法论层面的总开关。

## 十二诡道（始计篇）

> "故能而示之不能，用而示之不用，近而示之遠，遠而示之近。利而誘之，亂而取之，實而備之，強而避之，怒而撓之，卑而驕之，佚而勞之，親而離之。**攻其無備，出其不意**，此兵家之勝，不可先傳也。"

| 诡道 | 含义 |
|---|---|
| 能而示之不能 / 用而示之不用 | 隐藏真实能力与意图 |
| 近而示之远 / 远而示之近 | 误导对方判断我之方位时机 |
| 利而诱之 / 乱而取之 | 以利引诱、乘乱夺取 |
| 实而备之 / 强而避之 | 敌实则我备、敌强则我避 |
| 怒而挠之 / 卑而骄之 | 激怒使其失措、示弱使其骄纵 |
| 佚而劳之 / 亲而离之 | 使其疲于奔命、离间其内部 |

收束于"**攻其无备，出其不意**"——一切诡道服务于在敌方意料之外、未防备之处发力。"不可先传"：诡道随机应变，无法事先定死。

## 在全书的延展

- [军争篇](../07-军争篇.md)："兵以诈立，以利动，以分合为变"——把"诡"升为机动的基本原则。
- [虚实篇](虚实.md)："形人而我无形""至于无形，至于无声"——示形造假以制造虚实差。
- [兵势篇](../05-兵势篇.md)："乱生于治，怯生于勇"——以假乱、假怯、假弱调动敌人（[奇正](奇正.md)的示形面）。
- [九地篇](../11-九地篇.md)："始如处女，后如脱兔"——先骗松懈再迅猛出击。

## 思想分寸

- "诡"不是阴谋诡计的炫技，而是**降低己方代价、服务"全胜"与"庙算"的工具**。孙子的"诡"始终被"算"（[五事七计](五事七计.md)）和"全"（[不战而屈人之兵](不战而屈人之兵.md)）所约束。
