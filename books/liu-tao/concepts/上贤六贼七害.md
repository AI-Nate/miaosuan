# 概念 · 上贤·六贼七害（用人避害）

_六韬核心概念 · 出 [文韬·上贤](../01-文韬.md)，参 [举贤](../01-文韬.md) · 入库 2026-06-15_

**一句话**：君主用人，正面要"**上贤下不肖**"，反面要识别并清除两类危害——**六贼**（六种败坏国政的行为/人）与**七害**（七种不可重用之人）。这是六韬一份相当尖锐的"反面用人清单"，与"举贤而无用贤之实"的告诫互为表里。

## 上贤（用人总则）

> "王人者，**上贤，下不肖；取诚信，去诈伪；禁暴乱，止奢侈**。"

## 六贼（伤国的六种行为）

1. 大兴宫室池榭游乐 → **伤王之德**
2. 民不事农桑、任气游侠、犯禁不从教 → **伤王之化**
3. 臣结朋党、蔽贤智、障主明 → **伤王之权**
4. 士抗志高节、外交诸侯、不重其主 → **伤王之威**
5. 臣轻爵位、贱有司、羞为上犯难 → **伤功臣之劳**
6. 强宗侵夺、陵侮贫弱 → **伤庶人之业**

## 七害（不可用的七种人，撮要）

1. 无智略而恃重赏尊爵、强勇轻战侥幸者 → 勿使为将
2. 有名无实、出入异言、掩善扬恶者 → 勿与谋
3. 朴身恶服、语无为以求名、言无欲以求利者 → 伪人，勿近
4. 奇冠伟服、博辩虚论、窜居诽时俗者 → 奸人，勿宠
5. 谗佞求官、轻死贪禄、以虚论说主者 → 勿使
6. 雕文刻镂、技巧华饰而伤农事者 → 必禁
7. 伪方异伎、巫蛊左道、幻惑良民者 → 必止

> 用人之失的总后果："可怒而不怒，奸臣乃作；可杀而不杀，大贼乃发；兵势不行，敌国乃强。"

## 配套：举贤要"按名督实"

[举贤篇](../01-文韬.md) 补足正面：举贤而无功，是因"有举贤之名而无用贤之实"，失在"好用世俗之所誉"；正法是"**按名督实，选才考能，令实当其名、名当其实**"——形名责实（近法家循名责实）。

## 与他书对读

- 六贼七害 ↔ 尉缭子 [十二陵](../../wei-liao-zi/07-十二陵篇.md) 的十二害、孙子/吴子将之过——都是"负面清单"式的识人智慧。
- "按名督实"↔ 尉缭子 [刑德](../../wei-liao-zi/concepts/刑德重刑连坐.md) 的形名、申不害/韩非循名责实。
