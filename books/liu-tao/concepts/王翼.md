# 概念 · 王翼·阴符阴书（参谋部 + 保密通信）

_六韬核心概念 · 出 [龙韬·王翼/阴符/阴书](../03-龙韬.md) · 入库 2026-06-15_

**一句话**：这是《六韬》最独特、其他六书都没有的"**军事管理/参谋/情报**"贡献——**王翼**设计了一个 72 人的将帅参谋部（按职能分工），**阴符/阴书**则是一套军事保密通信系统（符号密码 + 分割密钥）。六韬之所以被视为"体系最备"，很大程度因为这一组制度设计。

## 王翼：72 人参谋部（古代"司令部"编制）

> "凡举兵帅师，以将为命……将有股肱羽翼**七十二人**，以应天道。"

按职能分工（撮要）：
- **腹心 1**：潜谋应卒、总揽计谋。
- **谋士 5 / 天文 3 / 地利 3 / 兵法 9**：决策、天象、地形、军法。
- **通粮 4 / 奋威 4 / 伏鼓旗 3**：后勤、突击、号令与伪装。
- **股肱 4 / 通材 3 / 权士 3**：工程守御、外交应对、奇谋诡变。
- **耳目 7 / 爪牙 5 / 羽翼 4 / 游士 8**：情报、激励、宣传造势、间谍。
- **术士 2 / 方士 2 / 法算 2**：心理（鬼神惑敌）、医药、会计后勤。

要点：这是中国（乃至世界）最早的**系统化幕僚/参谋部编制**之一——把"将"从单一统帅扩展为一个职能完备的指挥团队，"备数如法……万事毕矣"。

## 阴符：符号密码（八等军情）

> "主与将，有**阴符**，凡八等"：大胜一尺、破军擒将九寸、降城八寸、却敌七寸、坚守六寸、请粮益兵五寸、败军四寸、失利三寸。

- 以**符的长短**暗代八种军情，只有主、将知其义；"敌虽圣智，莫之能识"。泄露或稽留者诛。这是一套**最小信息量的预置密码**（只传符、不传文，杜绝内容外泄）。

## 阴书：分割密钥（长信加密）

> 长而复杂的军情，符不能尽，则用**阴书**："**一合而再离，三发而一知**"——把一封信分成三段，分别由三人传递，三人各持一段、相参而不知全文。

- 三段须全部会合才能还原全意——古典版的**信息分割 / 门限密钥**（threshold secret-sharing）思想，比近代密码学早两千年。

## 与他书对读

- 七书中唯一系统设计**参谋编制 + 通信保密**的就是六韬龙韬——孙子有 [用间](../../sunzi/13-用间篇.md)（情报来源），尉缭子有 [号令编制](../../wei-liao-zi/concepts/制必先定什伍编制.md)（金鼓旗），但"幕僚分工 + 加密通信"是六韬独有的"军事管理"维度，最具现代组织/信息论意味。
