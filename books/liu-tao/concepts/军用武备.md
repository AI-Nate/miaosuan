# 概念 · 军用·武备（先秦武器装备大全）

_六韬核心概念 · 出 [虎韬·军用](../04-虎韬.md)，参 [军略](../04-虎韬.md) · 入库 2026-06-16_

**一句话**：《六韬·虎韬·军用》是先秦兵书中**最详尽的一份武器装备与工程器材清单**——开列了一支万人之军所需的战车、连弩、各式重兵器、障碍器材、渡河攻城器具及其规格数量。它体现了六韬"**器械素备**"的思想：装备的预先齐备，是用兵的物质前提。

## 装备清单（撮要，万人之军）

| 类别 | 代表器械（撮要） |
|---|---|
| 战车 | 武冲大扶胥 36 乘（24 人/车）、武翼大橹扶胥 72 具、大黄参连弩大扶胥 36 乘、扶胥冲车、辎车（电车） |
| 远射 | 强弩六千、绞车连弩、飞凫/电影（信号箭，昼绛缟夜白缟） |
| 重兵器 | 方首铁棓（天棓）、大柯斧（天钺）、方首铁锤（天锤）各 1200 枚 |
| 障碍 | 行马（木螳螂剑刃扶胥）、木/铁蒺藜、地罗、飞钩、天罗虎落 |
| 渡河 | 飞桥（转关辘轳）、飞江、天浮铁螳螂、天潢（天舡） |
| 营具/攻具 | 虎落柴营、环利铁锁、伐木大斧、棨钁、铁叉、大镰、椓杙大锤 |
| 人员 | 甲士万人、强弩六千、戟楯/矛楯各二千、巧手（工匠）三百 |

> 各器械多标注**长度、重量、数量**（如"大柯斧，刃长八寸，重八斤，柄长五尺以上，千二百枚"），近乎一份**制式装备规格表**。

## 思想：器械素备 + 寓兵于农

- **器械为本**：[军略](../04-虎韬.md) 篇强调"虑不先设、器械不备…不可以为王者之兵"；攻守渡险皆有专用器械，"**三军用备，主将何忧？**"
- **寓兵于农**（[龙韬·农器](../03-龙韬.md) 呼应）：天下安定时"战攻守御之具，尽在于人事"——农具即兵器（耒耜为蒺藜、马牛车舆为营垒），平时务农即战时备战。

## 价值与对读

- **军事技术史料**：这是研究先秦兵器、战车、攻城/渡河工程的第一手文献，其细致程度为七书之最——孙子等重谋略而略器械，六韬独存这份"装备账"。
- 与尉缭子 [制必先定·什伍编制](../../wei-liao-zi/concepts/制必先定什伍编制.md)（制度先于行动）相呼应：尉缭子重"制"，六韬重"器"，都是把胜利建立在**开战前的硬准备**上。
