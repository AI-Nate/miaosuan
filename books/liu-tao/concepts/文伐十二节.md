# 概念 · 文伐十二节（不战颠覆术）

_六韬核心概念 · 出 [武韬·文伐](../02-武韬.md) · 入库 2026-06-15_

**一句话**：《六韬·武韬·文伐》提出**十二条"以文（非军事手段）颠覆敌国"的政治谋略**——通过投其所好、离间、收买、腐蚀、麻痹敌方君臣，在不动刀兵的情况下瓦解其内部，"**十二节备，乃成武事**"。这是中国兵学中最系统的"政治战 / 不战屈人"操作手册。

## 十二节（撮要）

| 节 | 手段（撮要） |
|---|---|
| 一 | 因其所喜，顺其志（养其骄，待其生事） |
| 二 | 亲其所爱，以分其威（使其一人两心） |
| 三 | 阴赂左右，深得其情（内情外泄则国生害） |
| 四 | 辅其淫乐，广其志（厚赂珠玉、娱以美人） |
| 五 | 严其忠臣而薄其赂（离间君臣，置我代理） |
| 六 | 收其内，间其外（才臣外用、敌国内侵） |
| 七 | 厚赂以锢其心（收其左右、令之轻业、蓄积空虚） |
| 八 | 赂以重宝、与之共谋（"重亲"，必为我用） |
| 九 | 尊之以名、示以大势（微饰圣人，国乃大偷） |
| 十 | 下之必信、承意应事（取信后微收，时至若天丧之） |
| 十一 | 塞之以道（收其豪杰、纳智士勇士，党具则"塞之"） |
| 十二 | 养其乱臣、进美女淫声、遗良犬马以惑之劳之 |

> 收束："**十二节备，乃成武事。所谓上察天，下察地，征已见，乃伐之。**"——文伐到位，武功（军事）才水到渠成。

## 要点

- **文伐 = 武事的前奏**：不是替代战争，而是先以政治手段把敌国蛀空，再以军事收尾。次序：文伐（瓦解）→ 征兆已见 → 武伐（出兵）。
- **打击对象是敌之"君臣关系与内部团结"**：核心机理是制造"一人两心""君臣相疑""贤奸倒置"，让敌国从内部失能。
- **手段冷峻而务实**：贿赂、美色、离间、收买豪杰、扶植乱臣——与儒家德治形成强烈反差，是六韬"全胜不斗"（[武韬·发启](../02-武韬.md)）的阴谋一面。

## 与他书对读

- 孙子 [谋攻](../../sunzi/03-谋攻篇.md)"上兵伐谋，其次伐交"只给原则；**六韬文伐把"伐谋伐交"展开为十二条可执行的颠覆术**，是孙子原则的操作化。
- 与孙子 [用间](../../sunzi/13-用间篇.md)（情报）互补：用间是"知敌"，文伐是"乱敌"。
- 现代迁移：近乎"政治战 / 认知战 / 颜色渗透"的古典蓝本——研究混合战争者常引此篇。
